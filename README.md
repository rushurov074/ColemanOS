# ColemanOS
Keon Coleman themed WebOS

This WebOS features a notes/news article app with 3 articles about Keon Coleman. The last article is from https://www.buffalorumblings.com/authors/anthony-marino.

It also features a live Weather app for the weather in Buffalo, NY. It uses the API from Open-Meteo.

DISCLAIMER: AI WAS USED!
I used Gemini to explain the guide when it got confusing, explain errors in my code, and explain how to use an API in JavaScript.

This was made to publish as a project in Stardance for Hack Club, but it serves as a fun, goofy project.
